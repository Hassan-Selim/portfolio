# Portfolio (HTML + CSS + JS)

موقع بصفحة هبوط مع أقسام:
Home / About / Services / Projects / Blog / Contacts  
ودعم **عربي/English** + **Light/Dark** (افتراضيًا حسب نظام الجهاز) مع حفظ الاختيارات في `localStorage`.

## التشغيل محليًا

### خيار 1 (مستحسن): Live Server
- افتح المجلد في VS Code / Cursor
- شغّل إضافة Live Server على `index.html`

### خيار 2: سيرفر بسيط عبر Python

```bash
python -m http.server 5500
```

ثم افتح `http://localhost:5500`

## تعديل البيانات
- رقم الهاتف والبريد: داخل `index.html` في قسم `contact`.
- المشاريع/الخدمات/النصوص: داخل `index.html` أو من قاموس اللغات في `main.js`.
